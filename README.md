# vue-ngrok-sample

## Project setup
```
yarn install
```

## ngrok setup

 1. regsiter ngrok(https://ngrok.com/)
 1. get Tunnel Authtoken
 1. you replace a 'YOUR_TUNNEL_AUTHTOKEN' with your real token in 'ngrok_auth.js'
 1. run 'yarn ngrok_add_authtoken'

## run ngrok 
```
yarn ngrok
```

## run dev sever after ngrok running
```
yarn run serve
```
